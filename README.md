# mandiblesegmentation
CycleGAN model for segmenting Mandibles from Panoramic Dental X-rays

|          Hyperparameter                                           |        Value         |
|-------------------------------------------------------------------|----------------------|
| Number of epochs                                                  |          750         |
| Optimizer                                                         |         Adam         | 
| Activation function used in final layer of Generator function     |         tanh         |
